# ゆかの高校◎◎部！！～極彩月光列伝～（仮） ティザーサイト

## 概要
ゆる学園日常伝奇バトル世界観共有企画「ゆかの高校◎◎部！！～極彩月光列伝～（仮）」の告知用ティザーサイトです。
架空の地方都市「山田県由嘉野市」を舞台に、日常と非日常（逆世界）が交差するストーリーとキャラクター設定を展開し、それらの世界観をゆるく共有していく企画となります。
本リポジトリは、将来のポータルサイト公開に向けた**事前告知ティザー版**の静的サイトソースコードを含んでいます。

## 今後のスケジュール
- **2026.05**: ティザー版サイト公開
- その後、β版参加者向けDiscordサーバー公開、ポータルサイト（Laravel等想定）の開発へと順次移行予定。
